== 다른 작업으로 전환

NetApp Backup and Recovery UI에서 다른 워크로드로 전환할 수 있습니다.

.단계
. 콘솔 왼쪽 탐색에서 *보호* > *백업 및 복구*를 선택합니다.
. 페이지 오른쪽 상단에서 *작업량 전환* 드롭다운 목록을 선택합니다.
. 전환하려는 작업 부하를 선택하세요.
+
페이지가 새로 고쳐지고 선택한 작업 부하가 표시됩니다.
