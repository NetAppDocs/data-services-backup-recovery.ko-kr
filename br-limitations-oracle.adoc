= NetApp Backup and Recovery for Oracle Database 워크로드의 알려진 제한 사항
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
이 버전에서 작동하지 않거나 제대로 작동하지 않는 플랫폼, 장치 또는 기능은 다음과 같습니다.  이러한 제한 사항을 주의 깊게 읽으십시오.

NetApp Backup and Recovery의 Oracle Database 워크로드 비공개 미리 보기 버전에서는 다음 작업이 지원되지 않습니다.

* 오프라인 백업


Oracle Database 워크로드의 비공개 미리 보기 버전에서는 Oracle Database가 NFS, SAN 또는 ASM SAN을 사용하는 독립 실행형 배포로만 지원됩니다.
